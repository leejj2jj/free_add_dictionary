About Free Add Dictionary

프로젝트 소개

Free Add Dictionary는 사용자들이 자유롭게 단어를 추가하고 공유할 수 있는 온라인 사전 플랫폼입니다.

주요 기능

  • 단어 검색 및 조회
  • 새로운 단어 추가
  • 단어 수정 및 삭제
  • 문의하기

기술 스택

  • Spring Boot
  • Spring Security
  • Spring Data JPA
  • Thymeleaf
  • Bootstrap
